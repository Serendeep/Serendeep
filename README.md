# Serendeep Rudraraju

*Software developer who occasionally finishes side projects.*

### About

I build things with Rust, JavaScript, and whatever else gets the job done. Right now most of my time goes into [brew-rs](https://github.com/brew-rs), a from-scratch rewrite of Homebrew in Rust, and an L3 Stack architecture for legal document generation. Before that rabbit hole, it was Flutter apps and web dev.

Outside of shipping code, I'm interested in open source tooling, how package managers actually work, and the weird corners of computing history. I write about some of this at [blog.serendeep.tech](https://blog.serendeep.tech).
